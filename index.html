CloStat

Github Repository

CloStat provides users with clothing recommendations based on the up-to-date weather information. It is a virtual closet that let the user keeps track of the status of clothing (clean or worn) to allow a more accurate recommendation.

BrainStorm

In my Human-Computer Interaction (HCI) Programming Studio class (Cogs 121), me and three other classmates brainstormed ideads about what mobile web application to make that centered around the theme of interacting with real world data. We each brainstormed separately, then came together to pitch our ideas to each other. We all liked the idea of having something to narrow down on what clothes we should wear from day-to-day because California's ever-changing weather and temperature made it hard for us to wear something we like while still enjoying the outdoor.

Paper Prototype

My group used paper prototype to design what would later be called Clostat. We mapped out each of the screens that will be displayed when the users click different buttons on the web app.

Markit

Github Repository
Development Plan

Markit allow users to scan barcodes to compare the user reviews on store products to make a wiser purchase. Markit is designed to combat the struggle of struggling to find the best quality product from the massive amount of choices offered to the consumer in store. It brings the online ecommerce feature of reviewing products to the physical world.

BrainStorm

In my HCI Interaction Design class (Cogs 120), me and two other classmates brainstormed ideas about what mobile web application we can make that deals with everyday problems and satisfy the theme of Environmental Interaction, and we all thought of the problem we had while grocery shopping. As we walked down the aisle, the sheer amount of choices overwhelmed us. We thought of the different ways we shopped and decided to integrate the online shopping experience with real world grocery shopping.

Paper Prototype

We used paper prototype to design Markit. We mapped out all the different possible screens for our app.

We used InVision to improve our paper prototype design and made it a more realistic app with clickable parts.

WhereYouAt(WYA)

Github Repository

WYA helps people coordinate while trying to meet up for an event. Like other event apps, it allows the user to make an event and invite others. What sets it apart from other event app is that it lets people see where the other event attendees are at if they chose to show their location. This feature enables attendees to guide one another to the event in the case that they get lost or canno find one another.
